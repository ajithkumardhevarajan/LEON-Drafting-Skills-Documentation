# Reuters AI Assistant Skills Documentation

Interactive reference guide for all Reuters AI Assistant skills and capabilities.

## View Live Documentation

This page is hosted on GitHub Pages and provides a comprehensive overview of:

- **Generic AI Assistant** - Orchestration layer and intelligent routing
- **Urgent Skill** - Create 1-2 sentence breaking news urgents
- **Update Skill** - Update existing stories with new information
- **Blank Slate Skill** - Generate complete 500-word stories from sources
- **Archive Search** - Semantic search integration
- **Capabilities Matrix** - Feature comparison across all skills
- **AI Models** - Model selection and usage per skill

## Local Development

To view locally, simply open `index.html` in a web browser.

## Updates

To update the documentation:
1. Edit `index.html`
2. Commit changes
3. Push to repository
